# 🚀 GenAI Engineering Journey


## 🧭 What This Repo Covers

A structured path from programming fundamentals to building real Generative AI systems:

- **Python Fundamentals** — core syntax, data structures, logic
- **Data Analysis** — Pandas, NumPy, data cleaning, feature engineering
- **Machine Learning** — core ML concepts and practical model building
- **LLM Applications** — prompt engineering, working with LLM APIs
- **RAG & Vector Databases** — retrieval-augmented generation systems
- **AI Agents** — building autonomous and semi-autonomous agent workflows
- **GenAI Systems** — end-to-end applied projects combining all of the above

Each topic folder contains hands-on exercises and notes — not copy-pasted tutorials.
